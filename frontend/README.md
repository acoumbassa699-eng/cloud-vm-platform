# Cloud VM Frontend

Next.js dashboard for Cloud VM Platform.

## Features

- User authentication (register, login)
- Instance management
- Real-time monitoring
- Billing and invoicing
- Project management
- Responsive design

## Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Environment Variables

```
NEXT_PUBLIC_API_URL=http://localhost:3001/api/v1
```

## Build

```bash
npm run build
npm start
```
